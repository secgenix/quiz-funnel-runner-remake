## Детальный план работ

### Этап 1: Telegram-бот (16-24 часа)

#### 1.1 Базовая структура бота (4-6 ч)
- [ ] Установка и настройка `python-telegram-bot` или `aiogram`
- [ ] Создание основного цикла бота
- [ ] Обработка команды `/start`
- [ ] Обработка текстовых сообщений (URL)
- [ ] Обработка списков URL (мульти-ссылки)
- [ ] Валидация URL перед обработкой

#### 1.2 Интеграция с раннером (4-6 ч)
- [ ] Интеграция `run_funnel()` с ботом
- [ ] Асинхронный запуск воронок
- [ ] Передача статуса в бот
- [ ] Обработка отмены задачи пользователем

#### 1.3 Система уведомлений (3-4 ч)
- [ ] Уведомление о начале обработки
- [ ] Прогресс: "Обработано N из M"
- [ ] Итоговый отчёт по завершении
- [ ] Уведомления об ошибках

#### 1.4 Управление задачами (5-8 ч)
- [ ] Очередь задач на обработку
- [ ] Статусы задач (pending, processing, completed, failed)

- [ ] Хранение состояния задач (SQLite/JSON)

## Технические рекомендации

### Стек технологий
```
Telegram Bot: aiogram 3.x (асинхронный)
Google Drive: google-api-python-client + google-auth
Google Sheets: gspread
Captcha: 2Captcha API / CapSolver
AI: OpenAI GPT-4 Vision API (или Anthropic Claude)
Database: SQLite (для состояния задач)
Queue: asyncio.Queue + ThreadPoolExecutor
```

### Архитектура
```
bot.py              # Telegram бот, интерфейс
runner.py           # Основной раннер воронок (рефакторинг main.py)
drive_uploader.py   # Google Drive интеграция
sheets_handler.py   # Google Sheets интеграция
captcha_solver.py   # CAPTCHA solving
classifier.py       # Улучшенная классификация экранов
ai_fallback.py      # AI fallback логика
models.py           # Модели данных, Task статусы
config.py           # Конфигурация
```
